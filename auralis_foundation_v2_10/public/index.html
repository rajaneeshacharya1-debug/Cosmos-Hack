Final-version notes
Static domain
Current stable demo URL can point to this laptop server through ngrok. Final product should use hosted backend.
Web app limit
A browser/PWA can show the dashboard, but it cannot reliably receive SMS, run MacroDroid actions, turn mobile data on, or execute commands while locked.
Android agent
Final protected-device control should be a native Android foreground service. The web UI remains excellent for the tracker dashboard.
